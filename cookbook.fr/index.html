Web.py Cookbook

Autres langages : japan 日本語 | chinese 简体中文 | english | ...

Documentation de type Cookbook pour web.py 0.3. Veuillez noter que certaines de ces fonctions ne sont pas disponibles dans les versions précedentes. La version actuelle 0.3 est la branche développement.

Mise en forme

  1. En termes de mise en forme, essayez d'utiliser un format de type cookbook... Comme ceci:

    Probleme: Vous voulez accéder aux données de la base ...

    Solution: Utilisez ce code ...

  2. Notez que les urls n'ont pas besoin de "web" dans leur structures -- juste "/cookbook/select" , et pas "/cookbook/web.select".

  3. Cette documentation est finalement pour la version 0.3, donc veuillez ajouter du code qui ne fonctionne qu'avec cette version.
